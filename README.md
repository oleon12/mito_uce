# Mito - UCE

This is a brief guide or tutorial on obtaining mitochondrial DNA, whether it is complete genomes or genes from UCE data. For this guide to work, your UCE data must be enriched during sequencing.

Also, to reduce mistakes and errors, I strongly suggest organizing your data in the following structure, so you do not need to change a lot of the scripts.

```
+ main
  +---- BASH_SCRIPTS
  +---- CONFS
  +---- ERR_OUT
  +---- LOG
  +---- raw_data
  +---- references
  +---- results

```
